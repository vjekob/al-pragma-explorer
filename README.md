# AL Pragma Explorer README

## Features

This extension enables a view in the Explorer.

The view shows a treeview of 
- workspaces
  - pragmas
    - files
      - Line `where-used`


![al-Pragme-View](https://i.imgur.com/WCWAdqx.gif)

## Requirements

Extension gets activate on AL language detection
